# ToDo List

Проект представляет собой веб-приложение ToDo-лист, где пользователь может создавать, редактировать и удалять задачи, отмечать их как выполненные, фильтровать, сортировать, искать по названию и дате, а также менять порядок задач с помощью drag-and-drop. Реализован полностью на HTML, CSS и JavaScript без использования фреймворков.

---

## Структура проекта

### `index.html`

Главная страница приложения. Содержит:

* подключение стилей и скриптов;
* всё остальное создаётся динамически через JavaScript.

### `css/style.css`

Основной файл стилей, задаёт визуальное оформление элементов страницы, шрифты, цвета, расположение блоков, адаптивность через flex и grid.

### `js/ui.js`

Содержит логику создания DOM-структуры страницы:

* заголовок приложения;
* форму добавления задач;
* панель управления (поиск, фильтры, сортировка);
* список задач.

### `js/logic.js`

Содержит всю логику работы приложения:

* добавление, удаление и редактирование задач;
* отметка задачи как выполненной;
* фильтрация, поиск и сортировка задач;
* drag-and-drop для изменения порядка задач;
* рендеринг списка задач на странице.

### `js/storage.js`

Содержит функции для работы с localStorage:

* сохранение задач;
* загрузка задач при старте приложения.

### `js/main.js`

Инициализация приложения после загрузки страницы:

* создание интерфейса через `ui.js`;
* загрузка сохранённых задач из `localStorage`;
* запуск логики приложения через `logic.js`.

---

## Реализованный функционал

* Добавление новых задач с указанием названия и даты;
* Удаление и редактирование существующих задач;
* Отметка задачи как выполненной с визуальным оформлением;
* Сортировка задач по дате;
* Фильтрация задач по статусу (выполненные/невыполненные);
* Поиск задач по названию;
* Изменение порядка задач с помощью drag-and-drop;
* Сохранение задач в `localStorage` и восстановление при перезагрузке страницы;
* Адаптивный интерфейс для различных размеров экранов.
